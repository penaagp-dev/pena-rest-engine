# PENA FRAMEWORK
Microservice specialist framework

## Getting Started
### Environtment Setup
```
mv env.example .env
```

### Installing
#### Contribution Mode
```
pip install -e .
```

#### Programmer Mode
```
pip install .
```

### Running
```
pena http serve
```

help
```
# pena -h
---------------------------------------------------------------------------------------------------

  pena <command> [<args>...]

Options:

  -v, --version                          Print version information and quit

Commands:
```
